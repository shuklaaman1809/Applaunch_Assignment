# User Management App

## Tech Stack
- Kotlin
- Jetpack Compose
- MVVM Architecture
- Room Database
- Retrofit
- Coroutines + Flow
- Hilt Dependency Injection

## Features
- Fetch users from API
- Store users locally using Room
- Display users from DB only
- Delete & Edit user
- Automatic UI updates using Flow
- Proper loading & error handling

## Architecture
Single Source of Truth (Room)
Network → Repository → Room → UI

## Setup
1. Clone repo
2. Open in Android Studio
3. Sync Gradle
4. Run app
